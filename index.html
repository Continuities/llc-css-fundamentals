#Let's dive into CSS
##What is CSS? HTML defines *content*, CSS defines the *presentation*. CSS is made up of rules that reference HTML elements (like `p` and `ul`) and define what those elements should look like. A CSS rule is made up of a *selector* and one or more *declarations*. h2 { color: red; font-size: 12px; } In the above example, `h2` is the selector and will apply all the styles in between the *declaration block* to all the `h2` headings.
##CSS Syntax A declaration tells the browser what style to apply to the selected element. A declaration is always made up of two parts: a property and a value. A property tells the browser what we are modifying (ex. font, colour, size). A value tells the browser how to modify the property (ex. Arial, green, small). ``` h2 { font-size: 12px; } ``` In the above example, which is the **selector**, **declaration**, **property** and **value**?
